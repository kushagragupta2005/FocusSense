🚀 FocusSense

FocusSense is a privacy-first focus tracking tool that helps users improve productivity using behavioral signals like typing activity, idle time, and app switching — all processed locally without compromising user data.



📌 Overview

FocusSense runs silently in the background and evaluates user focus levels in real-time. It provides subtle, non-intrusive feedback to help users stay on track without creating stress or distraction.



🎯 Main Goals
	•	Monitor user focus levels ethically and non-intrusively
	•	Provide useful feedback without disturbing the user
	•	Ensure complete privacy using fully offline processing
	•	Help users improve time management and self-discipline
	•	Maintain a simple, minimal interface based on HCI principles



⚙️ Features

🧠 Behavioral Tracking
	•	Typing Rhythm → Measures engagement via typing speed and patterns
	•	Idle Time → Detects inactivity indicating distraction or fatigue
	•	App Switching → Tracks context switching that reduces focus

📊 Real-Time Focus Score
	•	Combines signals into a weighted focus score
	•	Updates continuously in real time

🎨 Visual Feedback System
	•	🟢 Green → Deep focus
	•	🟡 Yellow → Moderate focus
	•	🔴 Red → Distracted

🔔 Smart Notifications
	•	Gentle alerts when focus drops
	•	Break suggestions after long sessions

📊 Requirement Discovery

User research was conducted via surveys and interviews with students and professionals at VIT and beyond.

Key Insights:
	•	Frequent distractions due to app switching and social media
	•	Strong concern for privacy (especially camera-based tools)
	•	Preference for passive, background tools
	•	Need for non-stressful, gentle feedback
	•	Typical focus span: 20–60 minutes
